# 📚 Book Management REST API

This project is a simple REST API built using Node.js and Express. It allows users to perform basic CRUD (Create, Read, Update, Delete) operations on a list of books. The data is stored in memory, so no database is required.

---

## 🚀 Features

- View all books  
- Add a new book  
- Update an existing book  
- Delete a book  

---

## 🛠️ Technologies Used

- Node.js  
- Express.js

## 📌 API Endpoints

| Method | Endpoint        | Description        |
|--------|----------------|--------------------|
| GET    | /books         | Get all books      |
| POST   | /books         | Add a new book     |
| PUT    | /books/:id     | Update a book      |
| DELETE | /books/:id     | Delete a book      |

## 💡 Project Overview

This API demonstrates how a simple backend system works using Express. It handles client requests and responses efficiently and performs operations on an in-memory data structure without using any database.

It helps in understanding how real-world backend applications are structured at a basic level.

---

## ⚠️ Important Note

- This project uses in-memory storage only  
- All data will reset when the server restarts  
- Suitable for learning and practice purposes
